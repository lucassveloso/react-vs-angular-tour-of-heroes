# React - Tour of Heroes
Projeto iniciado com base no boilerplate [react-etalpreliob](https://github.com/hyanmandian/react-etalpreliob).

### Development server
Rodar `yarn start`. A aplicação irá abrir sozinha no link `http://localhost:3000/`. 

### Build
Rodar `yarn build`. O [Webpack Bundle Analyzer](https://github.com/webpack-contrib/webpack-bundle-analyzer) estará disponível no link `http://localhost:8888/`

### Storybook
Rodar `yarn storybook`. O [Storybook](https://github.com/storybooks/storybook) estará disponível no link `http://localhost:6006/`
